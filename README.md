# lab-prompts

探索性 Prompt 实验仓库。收录在 AI 交互中沉淀出的模式化 Prompt 片段，用于验证、迭代和复用。

使用 [vasmc](https://github.com/glenzli/lync) 进行多语言编译管理。源文件位于 `prompt-src/`，编译产物输出到 `prompt/`。

## 目录

| 模块 | 说明 |
|------|------|
| [emergence](./prompt-src/emergence/) | 捕获对话中涌现的隐式协议、偏好与新概念，主动进行 prompt 化提炼 |
